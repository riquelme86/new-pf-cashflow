# PF 캐시플로우 SaaS 프로젝트

이 폴더를 Obsidian에서 볼트로 열어서 사용하세요.

## 설정 방법

### 1. Obsidian에서 볼트 열기
1. Obsidian 실행
2. "Open folder as vault" 선택
3. 이 폴더(`new-pf-cashflow`) 선택

### 2. Git 플러그인 설치 (선택사항)
1. Settings → Community Plugins
2. "Obsidian Git" 검색 후 설치
3. 플러그인 활성화

### 3. 자동 동기화 설정
- Git 플러그인이 설치되면 자동으로 깃허브와 동기화됩니다
- 변경사항이 있을 때 자동 커밋/푸시 가능
