## 🚀 Phase 2: MVP 개발 (3-4월)

### 백엔드 개발
```python
# 핵심 API 엔드포인트
/api/pf/calculate     # PF 지표 계산
/api/cashflow/monthly # 월별 현금흐름  
/api/sensitivity/run  # 민감도 분석
```

### 프론트엔드 개발
- **대시보드**: 핵심 지표 표시
- **입력 폼**: 프로젝트 기본 정보
- **차트**: 현금흐름 시각화
- **테이블**: 월별 상세 데이터

### 기술 스택 상세
- **Frontend**: Next.js 15 + TypeScript + Tailwind
- **Backend**: Next.js API + Python 계산 엔진
- **DB**: PostgreSQL (Supabase)
- **차트**: Recharts
- **배포**: Vercel
